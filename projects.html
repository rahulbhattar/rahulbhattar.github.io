PROJECTS

  1. E-Commerce

    This is a responsive e-commerce web application built with React. It features a dynamic product list sourced from JSON, advanced filtering by category and price, and a fully functional shopping cart with persistent storage using localStorage.

    • Designed and implemented responsive UI with reusable components using React and integrated category and price-based filtering logic for efficient product navigation.
    • Built cart functionalities: add, remove, update quantity, and persist using localStorage.
    • Managed global state using useContext to improve component communication.
    • Ensured user-friendly experience with synchronized state between product list and cart.
