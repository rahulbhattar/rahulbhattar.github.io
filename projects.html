PROJECTS

  1. Tic-Tac Toe

    I developed a classic Tic Tac Toe game using React, focusing on clean UI and component-based architecture. The game features real-time player interaction, win/draw detection logic, and a responsive design. It was a great way to apply React state management and conditional rendering while reinforcing core JavaScript logic.

    • Developed a fully functional Tic Tac Toe game using React, applying modern frontend practices and a component-based architecture.
    • Implemented game logic to handle player turns, win/draw detection, and game completion conditions.
    • Designed a clean and responsive user interface compatible with both desktop and mobile devices.
    • Utilized React hooks like useState to manage game state and update the UI dynamically.
    • Added game controls, including a reset button to allow users to easily restart and replay the game.
